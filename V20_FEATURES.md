# Titan V20 Features

## Stability
- Runtime error capture from Admin, Student, and Parent pages.
- Admin Error Monitor with resolve and cleanup actions.
- Firebase/Auth/Service Worker/LocalStorage health panel.

## Backup
- Admin can export a JSON backup for core tenant data.
- Admin can import and restore a compatible backup after confirmation.

## Smart Learning
- Student Timeline records page visits, exam events, notes, and certificate events.
- Student lesson notes stored per lesson.
- Video progress logging for lesson videos.
- Exam Pro timer and autosave for radio answers.

## Certificates
- Admin creates certificates for students.
- Student and Parent can view/print certificates.
- Each certificate includes a QR verification URL.

## Smart Reminders
- Admin can generate reminders for inactive students, debts, and expiring subscriptions.
- WhatsApp action links are generated when a phone number exists.

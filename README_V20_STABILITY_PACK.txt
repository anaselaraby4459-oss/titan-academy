Firebase Rules:
Open Firebase Console > Realtime Database > Rules, paste firebase_database_rules_saas_v20_stability_pack.json, then Publish.
Do not upload the rules file as a web page.

After uploading:
1. Press Ctrl + F5.
2. If old files still load, open DevTools > Application > Service Workers > Unregister.
3. Open the site again with ?tenant=titan.
